# Ephemeral File Sharer

A backend project to upload files that self-destruct after a set time.

## Features
- [ ] File Upload (Multipart/Form-Data)
- [ ] Unique Download Links
- [ ] Auto-deletion (Cron jobs)
- [ ] Resumable Uploads (TUS Protocol)
